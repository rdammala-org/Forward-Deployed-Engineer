## 🌓 Theme Support

- **Dark Mode** (Default for FDE portfolio)
- **Light Mode** (Manual toggle)
- **System Preference Detection** — Respects `prefers-color-scheme` media query
- **User Override** — Manual theme preference persists in localStorage

### Theme Implementation
- Color scheme uses CSS custom properties
- `data-theme` attribute on `<html>` element controls active theme
- `matchMedia('(prefers-color-scheme: dark)')` detects system preference
- localStorage stores user preference (manual toggles)
